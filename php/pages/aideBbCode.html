Aide : Le BBCode

Écrire en gras : [b]Votre texte[/b]
Donne : Votre texte

Écrire en italique : [i]Votre texte[/i]
Donne : Votre texte

Souligner : [u]Votre texte[/u]
Donne : Votre texte

Barrer : [s]Votre texte[/s]
Donne : Votre texte

Changer la couleur : [color=red]Votre texte[/color]
Donne : Votre texte

Changer la taille du texte : [size=18]Votre texte[/size]
Donne : Votre texte

Changer la police d'écriture : [font=roman]Votre texte[/font]
Donne : Votre texte

Insérer un lien : [url]https://mkpc.malahieude.net[/url]
Donne : https://mkpc.malahieude.net
OU [url=https://mkpc.malahieude.net]Le meilleur jeu du monde ![/url]

Masquer un message : [spoiler]À la fin, Mario sauve la princesse[/spoiler]
Donne:
Spoiler [AfficherMasquer]
À la fin, Mario sauve la princesse


Traduire un message : [fr]Message en français[/fr][en]Message in English[/en]
Donne:
Message en français
Message in English


(*) Ici, vous devez entrer l'adresse URL de l'image. Pour voir l'adresse URL d'une image, faites un clic droit dessus et choisissez "Copier l'adresse de l'image".
Si l'image se trouve sur votre ordinateur, vous devrez passer par un hébergeur d'images tel que PostImage. Cliquez sur "Choisissez image" et sélectionnez votre fichier.
L'URL de l'image est alors dans "Lien direct".
